# 自研滑轮日期时间选择器 设计

> 日期：2026-07-07
> 主题：用自研 iOS 式滑轮选择器替代原生 datetime-local

## 背景

原生 `<input type="datetime-local">` 无法限制分钟选项、UI 与页面风格不一致、跨浏览器行为差异大。决定自研一个 iOS 式滑轮选择器，实现：
- 滚轮交互（位置固定、数字滚动）
- 分钟只列 5 的倍数
- 不显示年份，跨年自动处理
- UI 与页面风格一致

## 目标

- 替代原生 datetime-local，用于新建和编辑两个场景
- 4 列滑轮：月 / 日 / 时 / 分
- 鼠标滚轮 + 拖拽 + 点击 三种交互
- 跨年自动判定（今年/明年）

## 非目标

- 不支持触摸手势（YAGNI，桌面优先）
- 不做年份列（跨年自动处理）
- 不做秒级精度

## 设计

### 一、组件接口

新建 `js/datetime-picker.js`，暴露全局对象 `DateTimePicker`：

```js
// 弹出模态选择器，返回选中的值（YYYY-MM-DDTHH:MM）或 null（取消）
DateTimePicker.open({
  value: '2026-07-10T15:30',  // 初始值（可选，默认"下一个整5分钟"）
  onConfirm: (value) => {},    // 确认回调
  onCancel: () => {}           // 取消回调（可选）
});
```

### 二、滑轮结构（4 列）

```
┌─────┬─────┬─────┬─────┐
│ 月  │ 日  │ 时  │ 分  │
├─────┼─────┼─────┼─────┤
│ 6   │ 9   │ 14  │ 25  │  淡出
│ 7   │ 10  │ 15  │ 30  │  淡出
│ 8   │ 11  │ 16  │ 35  │  ← 高亮选中
│ 9   │ 12  │ 17  │ 40  │  淡出
│ 10  │ 13  │ 18  │ 45  │  淡出
└─────┴─────┴─────┴─────┘
       [取消]      [确认]
```

- **月**：1-12
- **日**：1-31（按选中年/月动态调整，闰年 2 月 29 天）
- **时**：0-23
- **分**：00/05/10/.../55（仅 5 的倍数，共 12 项）

### 三、交互

| 操作 | 行为 |
|------|------|
| 鼠标滚轮在某列上滚动 | 该列上下滚动一格，停止后吸附 |
| 鼠标按住该列拖拽 | 跟随拖拽，松手吸附 |
| 点击该列某项 | 直接跳到该项 |

**位置固定，数字滚动**：每项固定高度（40px），可见 5 行，中间行（第 3 行）高亮为选中值。用 `transform: translateY(-offset)` 控制位移，`transition` 做吸附动画。

### 四、跨年规则（内部计算）

滑轮只让用户选月日时分，不暴露年份。系统按以下规则确定年份：

| 用户选的月日 | 落到的年份 |
|------------|-----------|
| 在今天之后 | 今年 |
| 在今天之前 | 明年 |
| = 今天 | 今年；若时分也已过，则明年 |

内部仍存完整 ISO 时间 `YYYY-MM-DDTHH:MM`，只是 UI 不暴露年份。

### 五、卡片显示规则（timeline.js 调整）

| 情况 | 显示 |
|------|------|
| 今年日期 | `M/D HH:MM`（如 `7/10 15:30`） |
| 明年日期 | `M/D HH:MM · 明年`（如 `3/15 10:00 · 明年`） |

逾期、排序、分组逻辑不变（基于完整 ISO 时间）。

### 六、接入方式

**新建场景**：
- `index.html`：原生 `<input type="datetime-local">` 替换为只读显示框 `.due-time-display` + 隐藏 `<input type="hidden" id="dueTimeInput">`
- `app.js`：点显示框 → `DateTimePicker.open()` → 确认后写入隐藏 input 并更新显示框文本

**编辑场景**：
- `timeline.js`：编辑框的 datetime-local 同样替换为只读显示框
- 点击 → `DateTimePicker.open()`，确认后写入内部状态

**移除**：
- `toDatetimeLocalValue()` 函数（不再需要格式转换）
- 所有 `showPicker`、`step`、原生 datetime-local 相关逻辑

### 七、文件改动点

| 文件 | 改动 |
|------|------|
| `js/datetime-picker.js` | **新建**：滑轮选择器组件（含 modal 浮层） |
| `css/style.css` | 新增滑轮、modal、显示框样式 |
| `index.html` | 替换原生 input；引入新 js |
| `js/app.js` | 新建场景接入选择器 |
| `js/timeline.js` | 编辑场景接入；跨年显示；移除 toDatetimeLocalValue |

### 八、不动的部分

- store.js（已无规整，无需改）
- 排序、分组、逾期判定、批量删除、emoji、导入导出
